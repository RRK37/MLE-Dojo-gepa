What is GEPA?

GEPA (Genetic Pareto) is a cutting-edge system optimization framework that uses reflective text evolution to automatically improve LLM agent performance through multi-objective optimization. Quick Start

Installation

# Clone the repository
git clone https://github.com/RRK37/MLE-Dojo-gepa.git
cd MLE-Dojo-gepa

# Set up environment
conda create -n mle-dojo-gepa python=3.11
conda activate mle-dojo-gepa
pip install uv
uv pip install -e .

# Install GEPA
pip install gepa

Basic Usage

# Run with GEPA optimization
python main.py \
    --competition-name titanic \
    --agent-type aide \
    --optimization gepa \
    --population-size 5 \
    --generations 10 \
    --objectives score,time,complexity

Configuration

# config.yaml
gepa:
  population_size: 5
  generations: 10
  mutation_rate: 0.15
  crossover_rate: 0.8
  objectives:
    - name: leaderboard_score
      weight: 0.6
      maximize: true
    - name: execution_time
      weight: 0.2
      maximize: false
    - name: code_complexity
      weight: 0.2
      maximize: false
  
pattern_analysis:
  enabled: true
  analysis_frequency: 3  # Analyze every 3 steps
  min_solutions: 3       # Minimum solutions to analyze
